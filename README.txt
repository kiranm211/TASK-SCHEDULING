****TASK SCHEDULING****
THE MAIN AIM IS TO PERFORM MAX NO. OF TASKS TO GET THE MAX BONUS 
NOTE:
->EACH TASK TAKES 1 DAY TO COMPLETE
->THE MAX NO. OF TASKS DEPENDS ON THE DEADLINE PROVIDED AS INPUT

STEPS TO FOLLOW:-

*OPEN A C COMPILER
*COMPILE THE FILE TASK SCHEDULING.c 
*PROVIDE THE INPUT IN THE GIVEN FORMAT
	->NO. OF TASKS
	->DEADLINE AND BOUNS IN N LINES 
*RESULTS WILL DEPEND ON THE INPUTS
